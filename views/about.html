About

  • What is N0rthWestW1nd?

    • N0rthWestW1nd is a Discord bot coded by NorthWestWind#1885 with the npm module "discord.js".
    The bot contains a ton of commands. From the basic ones like "role", "kick" and "ban", to gaming APIs like "minecraft".

    The name of the bot is simply come from the author's name, but replaced "o" to "0" and "i" to "1".

  • How to use the commands?

    • Since there are a lot of commands, I wrote a manual to help you out!

    To get the manual, simply invite N0rthWestW1nd to your server.
    Then, type "?help" in any channel and you will received a copy of the manual!
    How convenient!

    Some of the commands have their very own subcommands.
    For any question about subcommands, please refer to the manual.

  • DLC

    • There is actally a DLC for N0rthWestW1nd!
    (Don't worry. It's completely free!)

    What the DLC do is add a leveling system to your server!
    The idea was inspired by MEE6.

    All datas are stored in a SQL database and we constantly get backups for it.
    So, don't worry about data loss!
